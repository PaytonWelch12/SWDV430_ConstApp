<h1>OOP/Design Pattern/Java - Practice</h1>


<h2>Description</h2>
This simple Java app, developed for learning Java, OOP, and design patterns, manages files for a fictional construction company. It organizes files into specific classes that implement a file base class. It then uses a visitor pattern to access the data from these classes and then print them in an orderly fashion.
